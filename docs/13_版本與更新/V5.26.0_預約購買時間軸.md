Version: V5.26.0  
Date: 2026-07-07  
Status: Completed

---

## 1. 修改原因

使用者明確指定想要的格式為依時間排序的購買／預約表，例如：

> 2026/11 買機票 EVA Air BR178／BR129 36000 TWD  
> 2026/12 續約洲際酒店大使 250 USD  
> 2027/9/17 搶購 USJ Express Pass 7

因此本版新增正式「預約與購買時間軸」，作為出發前付款、續約、搶票、訂位與申請事項的主控表。

| 項目 | 變更 |

| 機票 | NT$35,000 → NT$36,000／兩人。 |
| 交通預算 | NT$54,664 → NT$55,664。 |
| 建議版主預算 | NT$324,664 → NT$325,664／兩人。 |
| 每人平均 | NT$162,332 → NT$162,832。 |
| 安全上限 | 維持 NT$380,000／兩人。 |
| Ambassador 續約 | USD 250，外幣獨立追蹤，不硬轉 TWD。 |
| 精品購物 | 仍另計，不混入基本旅費。 |

## 7. 結論

V5.26.0 起，專案正式增加一份依日期排序的預約與購買主控表。

目前 TWD 主控預算為：

> NT$325,664／兩人；安全上限 NT$380,000／兩人。

Ambassador 續約費：

> USD 250，獨立追蹤，待匯率或實際刷卡金額確認後再決定是否併入 TWD 預算。
