# V5.58.0 Web Threads Cover Style Refresh

Date: 2026-07-13
Status: Completed
Type: Website presentation refresh / no itinerary or budget decision change

## 修改目的

使用者提供 Threads 影片作為視覺參考，希望網站輸出更接近「深色封面 + 白底目次 + 細緻章節導覽」的旅行小冊子感。V5.58.0 針對 `web/` 重新整理首頁版型，但不改動資料庫與預算基準。

## 修改範圍

| 檔案 | 修改內容 |
|---|---|
| `web/index.html` | 重新編排首屏為深色封面與目次頁，新增倒數天數、旅程節奏條與索引式章節導覽。 |
| `web/styles.css` | 重寫視覺系統為 Threads 風格參考的封面／目錄／紙本感內容頁，並保留預算 JS 需要的樣式。 |
| `web/budget.js` | 新增封面倒數天數自動計算，其餘預算邏輯維持不變。 |
| `README.md` | 更新目前主控版本為 V5.58.0，並同步網站版本說明。 |
| `CHANGELOG.md` | 新增 V5.58.0 變更紀錄。 |

## 設計方向

- 首屏改成深青色封面，使用大字標題與狀態膠囊，對齊參考影片的海報感。
- 新增白底 `INDEX` 區塊，以編號列出行程、亮點、飯店、準備、預算、餐飲與備註。
- 內容區維持原有大阪旅遊主線，但整體排版改成更像一本對外分享的小冊子。
- 保留既有預算資料與預算計算結果，不新增平行版本。
