## 修改內容

新增 KKday / Klook 票券平台候補資料，包含：

- USJ Studio Pass / Express Pass 平台比價與備援策略
- KIX → 梅田 / 大阪站交通候補
- Osaka Amazing Pass 候補判斷
- Osaka e-Pass 候補判斷
- eSIM / SIM / Pocket Wi-Fi 採購時機
- 私人接送與行李配送候補
- 三星米其林餐廳不得以平台商品直接取代正式預約管道

---

## 修改原因

2027 大阪自由行需要在出發前持續追蹤 KKday / Klook 的票券、交通、通訊與便利服務，但不得因平台促銷或商品推薦直接改動正式行程。

因此本次更新採用 Candidate 方式：

1. 先建立 `database/` 資料卡。
2. 再同步 `docs/` 說明文件。
3. 更新 `WATCHLIST.md`。
4. 更新 `CHANGELOG.md`。
