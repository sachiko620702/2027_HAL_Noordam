# V5.6.0 - GitHub Pages Root Homepage Sync

Date: 2026-07-06  
Version: V5.6.0  
Status: Completed

---

## 修改內容

新增 GitHub Pages 根目錄首頁同步輸出：

- `scripts/daily_maintenance.py` 會在每日同步時生成根目錄 `index.md`。
- `scripts/daily_maintenance.py` 會同時維持 `docs/index.md` 與 `docs/14_自動同步狀態/每日同步狀態.md`。
- `PROJECT_RULE.md` 新增「每日維護變更後必須同步根首頁與 docs 首頁」的規則。

---

## 修改原因

使用者啟用了 GitHub Pages，但根目錄看起來沒有首頁，因此需要把可公開瀏覽的入口補齊，並且讓每日維護流程自動同步輸出到根目錄首頁。
