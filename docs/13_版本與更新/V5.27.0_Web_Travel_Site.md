# V5.27.0｜Web 旅遊網站集中到 `web/`

Date: 2026-07-07  
Version: V5.27.0  
Status: Completed

---

## 1. 修改內容

將對外公開的 GitHub Pages 風格網站集中到 `web/`：

- 新增 `web/index.html` 作為旅遊社顧客版首頁。
- 新增 `web/styles.css` 作為共用視覺樣式。
- 更新 `scripts/daily_maintenance.py`，讓後續同步維持根目錄與 `docs/` 首頁為入口頁。
- 將根目錄 `index.md` 與 `docs/index.md` 縮為入口頁，改成導向 `web/`。
- 將網站內容重新整理成旅行社給顧客看的版型，包含總覽、每日行程、行事曆提醒、預算摘要、餐飲亮點與待確認清單。

---

## 2. 修改原因

使用者希望把所有給 GitHub Pages 的檔案集中到 `./web`，並且從旅行社顧客視角重新設計頁面，使其更像正式提案頁而不是資料庫索引。
