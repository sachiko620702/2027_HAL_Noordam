## 修改內容

將 GitHub Pages 首頁升級為旅遊計畫介紹網站：

- `scripts/daily_maintenance.py` 的首頁模板改成更完整的 landing page 版型。
- 首頁新增旅程節奏、專案框架、每日追蹤指標、快速入口與 watchlist 區塊。
- `index.md` 與 `docs/index.md` 會同步輸出相同的網站式首頁。

---

## 修改原因

使用者希望專案內容不要看起來像 README，而是像旅遊計畫介紹網站，因此首頁需要有更完整的版面結構與視覺層次。
